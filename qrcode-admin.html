📖 使用说明

如何获取收款码

  1. 微信收款码：打开微信 → 我 → 服务 → 收付款 → 二维码收款 → 保存图片
  2. 支付宝收款码：打开支付宝 → 首页 → 收钱 → 保存图片
  3. 在本页面上传保存的二维码图片
  4. 点击"保存"按钮，二维码将存储在浏览器中
  5. 用户在支付页面即可看到您的收款码

⚠️ 注意事项

  1. 二维码数据存储在浏览器本地，更换设备需要重新上传
  2. 建议定期更换收款码，确保账户安全
  3. 图片格式支持 JPG、PNG，建议大小不超过 500KB
  4. 如需备份，可使用浏览器的导出功能

⚙️ 数据管理

导出功能可以将二维码数据保存为JSON文件，方便备份或在其他设备上恢复。
