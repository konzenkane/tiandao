## 数据存储结构（localStorage）
```javascript
// 用户系统
fortune_user_id              // 用户唯一ID
fortune_my_invite_code       // 自己的邀请码

// 邀请裂变
fortune_share_data_${id}     // 邀请数据
fortune_pending_inviter      // 待处理的邀请人
fortune_pending_invite_code  // 待处理的邀请码
fortune_invitation_records   // 邀请记录数组
fortune_already_invited      // 是否已被邀请过

// VIP与奖励
fortune_vip_data             // VIP状态（天数、解读券）
fortune_streak_rewards       // 连续打卡已领取奖励
fortune_checkin_rewards      // 打卡奖励记录

// 功能模块
tarot_history                // 塔罗牌记
fortune_checkin_${y}_${m}    // 每月打卡记录
```

## 核心流程

### 邀请裂变流程
```
用户A分享 → ?invite=CODE&inviter=ID
    ↓
用户B打开 → 保存pending到localStorage
    ↓
用户B完成运势测算 → 触发绑定 → 记录邀请关系
    ↓
用户A进度+1 → 解锁奖励 → 点击领取
```

### 连续打卡奖励
```
打卡 → 计算连续天数 → 检查里程碑
    ↓
7天 → 大师精批券 × 1
30天 → VIP月卡 × 1
    ↓
记录到本地 → 提示用户（体验版）
```

### 塔罗占卜流程
```
选择问题 → 冥想引导（30秒）→ 扇形牌阵
    ↓
点击抽牌 → 震动+音效+翻牌动画
    ↓
生成解读 → 保存牌记 → 分享图
```

### 六爻摇卦流程
```
输入问题 → 点击摇卦（6次）
    ↓
每次：铜钱动画 → 随机正反面 → 更新爻象
    ↓
六爻毕 → 生成本卦/变卦 → 显示案例库
```

## 技术亮点

1. **触觉反馈**：navigator.vibrate API
2. **Web Audio**：简单音效合成
3. **3D动画**：CSS transform-style: preserve-3d
4. **本地存储**：localStorage结构化数据
5. **邀请绑定**：URL参数 + localStorage状态机
